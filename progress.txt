## デザイン仕様: ブロブ背景アニメーション

### 概要
元サイト（https://careers.giftee.co.jp/）で使用されているインタラクティブな背景要素。
オレンジ色の有機的な球体がマウスの動きに追従し、形を変えながらアニメーションする。

### 実装ファイル
- `src/components/common/BlobBackground.tsx` - コンポーネント本体
- `src/components/common/BlobBackground.module.css` - スタイル

### 技術仕様

| 項目 | 詳細 |
|------|------|
| 描画方式 | HTML Canvas + requestAnimationFrame |
| 形状生成 | Perlin Noise アルゴリズム（FBM: フラクタルブラウン運動） |
| ブロブ数 | 2つ（メイン: 大、サブ: 小） |
| カラー | #e76746（コーラル）、#f5a080（ライトオレンジ） |
| ブラー効果 | filter: blur(80px) |
| 透明度 | opacity: 0.6 |
| マウス追従 | イージング係数 0.02 でスムーズに追従 |

### ブロブ設定
```
メインブロブ:
  - 初期位置: 画面右上 (70%, 30%)
  - 半径: min(width, height) * 0.35
  - 速度: 0.8

サブブロブ:
  - 初期位置: 画面左下 (30%, 70%)
  - 半径: min(width, height) * 0.25
  - 速度: 0.6
```

### アクセシビリティ
- `prefers-reduced-motion: reduce` 時はアニメーション無効化
- モバイルではブラー軽減（60px）、透明度低下（0.5）
