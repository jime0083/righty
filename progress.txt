# ライティ採用サイト クローン開発 - 進捗管理

## プロジェクト概要
- プロジェクト名: right
- 参照元: https://careers.giftee.co.jp/
- 技術スタック: Vite + React + TypeScript + CSS Modules

## ルール
1. 元サイトの色とフォントを厳守
2. 画像はダミーを配置（後で差し替え予定）
3. レスポンシブルールも完全にコピー
4. React SPA として構築
5. 社名は「ライティ」としてダミーテキストを使用

## 技術メモ

### 開発コマンド
```bash
# 開発サーバー起動
npm run dev

# ビルド
npm run build

# ビルドプレビュー
npm run preview

# 型チェック
npx tsc --noEmit
```

### フォルダ構造
```
right/
├── public/
│   └── images/          # ダミー画像（後で差し替え）
├── src/
│   ├── assets/
│   │   ├── images/      # ソース内画像
│   │   └── videos/      # ソース内動画
│   ├── components/
│   │   ├── common/      # Header, Footer, Button, Card
│   │   └── sections/    # Hero, About, Culture, Jobs, Environment, NewGraduate, Blog
│   ├── hooks/           # カスタムフック
│   ├── pages/           # ページコンポーネント
│   ├── styles/          # グローバルスタイル
│   │   ├── global.css
│   │   ├── variables.css
│   │   └── design-spec.md
│   └── utils/           # ユーティリティ
├── index.html
├── vite.config.ts
└── package.json
```

### GitHub Pagesデプロイ手順
1. vite.config.tsのbaseをリポジトリ名に合わせる
2. `npm run build`でdistフォルダを生成
3. GitHubリポジトリにプッシュ
4. Settings > Pages > Source: Deploy from a branch
5. Branch: gh-pages (またはmain/distフォルダ)
